# Contributing

Thanks for helping improve PC Health Check.

## Good First Contributions

- Improve report wording for non-technical users.
- Add tests for formatting helpers.
- Add localization files.
- Improve privacy redaction.
- Add documentation for common Windows performance issues.

## Project Principles

- Read-only by default.
- No telemetry.
- No network uploads.
- Avoid collecting sensitive personal data.
- Prefer clear plain text over clever output.
- Explain uncertainty instead of pretending the report is a final diagnosis.

## Pull Requests

Please include:

- what changed;
- why it helps users;
- what Windows version you tested on;
- before/after sample report output when report formatting changes.
