# Security Policy

## Reporting a Vulnerability

Please open a private security advisory on GitHub if available, or contact the maintainer through the repository profile.

## Data Handling

This tool is designed to run locally and write reports locally. It should not upload data or call external services.

Reports may include device and environment information. Review reports before sharing them publicly.

## Out of Scope

This project is not an antivirus, malware remover, forensic tool, or remote management system.
